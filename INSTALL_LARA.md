## 🚀 Способы установки

### 1️⃣ Через Scarlet (Самый простой)
1. Установи Scarlet: https://usescarlet.com
2. Открой https://liver-vintage-marco-thus.trycloudflare.com
3. Нажми **"Установить через Scarlet"**
4. Готово! Scarlet автоматически подпишет и установит

### 2️⃣ Через TrollStore (Если уже установлен)
1. Открой https://liver-vintage-marco-thus.trycloudflare.com
2. Нажми **"Установить через TrollStore"**
3. Готово! Установится навсегда без подписи

### 3️⃣ Через ESign
1. Установи ESign: https://esign.yyyue.xyz
2. Открой https://liver-vintage-marco-thus.trycloudflare.com
3. Нажми **"Установить через ESign"**
4. ESign подпишет с твоим сертификатом

### 4️⃣ Прямая загрузка (Требуется компьютер)
1. Скачай: https://github.com/rooootdev/lara/releases/latest/download/lara.ipa
2. Установи через AltStore или Sideloadly
