🔐 LARA - Online Signing

Подпиши IPA онлайн без компьютера

📥 Сначала скачай IPA:
https://github.com/rooootdev/lara/releases/latest/download/lara.ipa

1️⃣ ESign (Рекомендуется)

Подписывай прямо на iPhone без компьютера

  • ✅ Работает без компа
  • ✅ Бесплатно
  • ⚠️ Нужен свой Apple ID
  • ⏱️ 7 дней, потом переподписать
Открыть ESign Инструкция: Скачай ESign → Импортируй lara.ipa → Подпиши своим Apple ID

2️⃣ Scarlet

Альтернативный сервис подписи

  • ✅ Работает без компа
  • ✅ Простой интерфейс
  • ⚠️ Нужен Apple ID
Открыть Scarlet

3️⃣ AppDB (Платно)

Профессиональный сервис с сертификатами

  • ✅ Не нужен свой Apple ID
  • ✅ 1 год действия
  • ❌ Платно (~$20/год)
Открыть AppDB

4️⃣ SignTools (Свой сервер)

Если есть доступ к компу удалённо

  • ✅ Полный контроль
  • ✅ Бесплатно
  • ⚠️ Нужен запущенный сервер
GitHub SignTools
⚠️ Важно:
  • Используй только свой Apple ID
  • Не вводи пароль на подозрительных сайтах
  • ESign и Scarlet - проверенные сервисы
  • После подписи: Settings → General → VPN & Device Management → Trust
📖 Быстрая инструкция (ESign):
  1. Скачай ESign по ссылке выше
  2. Открой ESign → нажми "+" → Import
  3. Выбери скачанный lara.ipa
  4. Нажми "Sign" → введи свой Apple ID
  5. Дождись подписи → Install
  6. Settings → Trust certificate
  7. Готово! 🎉
